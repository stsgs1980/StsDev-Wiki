## Как пользоваться

1. Редактируешь любой `.md` файл в этой папке
2. Запускаешь `push.bat` — всё улетает в GitHub и GitBook

Если создаёшь **новый** файл — пропиши его в `SUMMARY.md`. Пример:

```
## Моя тема
* [Название статьи](папка/файл.md)
```

Если файла нет в `SUMMARY.md` — GitBook его не увидит.

## VS Code — полезные фишки

| Действие | Что даёт |
|---|---|
| `Ctrl+Shift+V` | Превью markdown — как будет выглядеть в GitBook |
| `Ctrl+P` → имя файла | Быстрый поиск по всей wiki |
| `.vscode/wiki.code-snippets` | Сниппеты — вставляй шаблоны в 2 клика |
| Git Automator | Авто-commit при сохранении — push.bat не нужен |
| `---` | Горизонтальная линия в markdown |
| `images/папка` | Картинки: `![](images/file.png)` |

> Если VS Code подчёркивает русский текст красным — отключи проверку орфографии или добавь русский словарь.
